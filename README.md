HETIC-P2020-10
========
Front-end project.

## Installation
```sh
# Clone the repository
git clone https://github.com/greeeg/hetic-p2020-10.git

# Go to project folder
cd hetic-p2020-10

# Install packages
yarn

# Install globally Eslint
npm i -g eslint eslint-plugin-import eslint-config-airbnb eslint-config-airbnb-base

# Run Gulp.js
gulp
```
